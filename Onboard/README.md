# 제목

## 소제목
: A self-improving robotic agent

### 기사내용 요약
Google DeepMind가 로봇팔을 통해 여러 작업을 수행하며 스스로 학습하는 RoboCat을 소개함.
실제 환경과 시뮬레이션 환경 모두에서 동작할 수 있는 멀티 모달 모델인 Gato(스페인어로 '고양이')를 기반으로 함.
